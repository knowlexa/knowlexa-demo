Knowlexa is an AI-powered knowledge assistant that allows organizations to ask questions directly on their internal documents.

- Source-based responses
- Fully private & open-source

## Tech Stack
- Streamlit
- Python
- FAISS
- Sentence Transformers
- HuggingFace Models
